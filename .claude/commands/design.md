Use the **frontend-design** skill AND **forth-ai-brand** skill together to build interfaces for Forth AI.

## Design Philosophy

**One input. Infinite outcomes.**

Like Google Search or ChatGPT, Forth AI interfaces should:
- Present a single, obvious entry point
- Interpret intent intelligently (polymorphic behavior)
- Hide complexity in the AI layer, not the UI
- Pass the 30-second test (new user completes task without training)

## Requirements

Apply the forth-ai-brand skill for:
- **Visual identity**: Forth Blue (#0066FF), Forth Orange (#FF6600), Playfair Display + Lora typography
- **Tech stack**: React 19, RTK Query v2.10, Tailwind CSS 4.1, Vite 7.2.2
- **Design principles**: Radical simplicity, universal interface pattern, joy as a metric

## Anti-Patterns to Avoid

- Feature-soup dashboards
- Wizard workflows (multi-step when single-step suffices)
- Explicit mode selection
- Confirmation dialogs (use undo instead)
- Generic AI aesthetics (Inter font, purple gradients)

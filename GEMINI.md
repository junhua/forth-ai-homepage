# Project: Forth AI Homepage

This file provides a summary of the project context for the Gemini agent.

## 1. Company Overview

- **Name:** Forth AI
- **Business:** Forth AI builds agent-native enterprise software to automate manual tasks and workflows (e.g., data logging, customer support, HR, finance).
- **Key Offerings:**
    1.  **Software Platform:** An agent-native platform for building and deploying AI agents.
    2.  **Agentic Workstation:** An affordable, on-premise hardware solution with fine-tuned models, ensuring data privacy and predictable costs.

## 2. Project Goal

To create a modern, professional homepage for Forth AI that effectively communicates its value proposition to potential enterprise customers.

## 3. Technical Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/UI
- **Deployment:** Vercel (via GitHub Actions)

## 4. Design & Content

- **Design Inspiration:** The aesthetic is based on `scale.com` - a dark, modern, professional, and data-centric theme.
- **Content Strategy:** The narrative is inspired by `adept.ai` - leading with a bold vision, followed by tangible demos and clear explanations of the technology and its benefits.
- **Live URL:** https://forth-ai-homepage.vercel.app/

## 5. Style Guide Summary

- **Theme:** Dark
- **Primary Background:** `#0A0A0A` (Near Black)
- **Accent Color:** `#0070F3` (Vibrant Blue)
- **Font:** Inter (Sans-Serif)

## 6. Design Philosophy Deep Dive

This section provides a more detailed comparison of the design inspirations for the project.

| Aspect | `scale.com` (Primary Aesthetic) | `adept.ai` (Narrative Inspiration) |
| :--- | :--- | :--- |
| **Theme** | Dark & Corporate | Dark & Futuristic |
| **Vibe** | Professional, Trustworthy, Data-Centric | Visionary, Innovative, Minimalist |
| **Layout** | Information-dense, structured sections | Open, spacious, focused on single concepts |
| **Visuals** | Abstract 3D graphics, data visualizations | Abstract graphics, subtle animations |
| **Our Choice** | We adopted **Scale's dark, professional aesthetic** to build trust with an enterprise audience, while using **Adept's vision-led narrative structure** for our content. |
